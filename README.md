# 霍格華茲模擬器

供 AI 專案使用的霍格華茲魔法學校沉浸式文字遊戲設定與腳本。將完整的提示詞系統、遊戲規則、介面元件整合於此，可直接載入至支援 AI 對話的環境中使用。

## 遊戲概覽

本模擬器為**長期沉浸式文字 RPG**，背景設定於哈利波特原著宇宙。玩家以 11 歲新生身份入學霍格華茲，體驗為期 7 年的魔法學院生活。

**核心體驗：**

- 校園日常：上課、考試、魁地奇、夜遊
- 友誼與感情線：依玩家行為自然推進，NPC 有獨立個性與秘密
- 魔法學習：循序漸進，涵蓋原著全課程
- 隱藏支線：圍繞霍格華茲古老秘密，隨探索逐步解鎖
- 家庭互動：假期日常與家庭背景對劇情的影響

## 使用方式

### 1. 載入設定

將 `src/conventions/` 下的所有 `.md` 檔案作為 System Prompt 或 Project Knowledge 載入 AI 對話環境。

### 2. 開始遊戲

在對話中輸入以下任一指令：

```
開始遊戲
開啟偉大的巫師之旅
/start
```

### 3. 角色建立

遊戲開始前會引導玩家填寫角色資訊，未填寫項目自動隨機生成：

| 類別 | 設定項目 |
|------|---------|
| 個人 | 姓名、暱稱、性別、生日、MBTI、性格傾向、外貌 |
| 家庭 | 出身（純血／混血／麻瓜）、家庭氛圍、家庭成員、寵物 |
| 入學 | 學院、魔杖、天賦興趣 |
| 劇情 | 隱藏支線開啟方式、與原著 NPC 的初始關係 |

## 遊戲系統

### 學院分

| 行為 | 效果 |
|------|------|
| 課堂表現優異、幫助同學、解謎、勇敢行為 | 加分 |
| 夜遊被抓、違規魔法、進入禁區、課堂搗亂 | 扣分 |

扣分不只是數字，會影響 NPC 態度、教授關注與後續事件觸發。

### 魔法學習

課程依學年循序漸進，包含原著全科目：符咒學、魔藥學、變形學、黑魔法防禦術、藥草學、天文學、占卜學等。施法時考量熟練度、情緒狀態、環境與違規風險。

### 時間推進

每天分為清晨、上午課程、午餐、下午課程、黃昏自由活動、夜晚、深夜特殊事件。時間自然推進，無需每次寫完完整一天。

## 存取進度

### 存檔指令

```
/save        # 手動存檔，選擇快速封印或完整封印
取出儲思盆    # 顯示儲思盆存檔管理介面
```

### 存檔格式

**快速封印**：精簡 JSON + 100 字劇情摘要

```json
{
  "character": { "name": "", "house": "", "year": 0, "wand": "", "origin": "" },
  "location": "",
  "npcs": { "NPC名": "關係簡述" },
  "flags": { "支線旗標": true },
  "notes": ""
}
```

**完整封印**：完整角色資料、NPC 好感度、支線旗標、當前時間地點 + 300 字詳細摘要

### 讀取存檔

在對話開頭貼上存檔 JSON，AI 將重建世界狀態並以魔法語氣確認：

> _記憶自儲思盆的銀霧緩緩旋轉，記憶自深處一幕幕浮現⋯_

### 自動存檔觸發時機

- 重大劇情節點（支線開啟、感情線轉折、重要選擇）
- 高風險行動前（夜遊禁林、與敵對角色衝突、魁地奇比賽）
- 每累積 10–15 輪對話
