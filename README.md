# 霍格華茲模擬器

供 AI 專案使用的霍格華茲魔法學校沉浸式文字遊戲設定與腳本。將完整的提示詞系統、遊戲規則、介面元件整合於此，可直接載入至支援 AI 對話的環境中使用。

## 遊戲概覽

本模擬器為**長期沉浸式文字 RPG**，背景設定於哈利波特原著宇宙。玩家以 11 歲新生身份入學霍格華茲，體驗為期 7 年的魔法學院生活。

**核心體驗：**

- 校園日常：上課、考試、魁地奇、夜遊
- 友誼與感情線：依玩家行為自然推進，NPC 有獨立個性與秘密
- 魔法學習：循序漸進，涵蓋原著全課程
- 隱藏支線：圍繞霍格華茲古老秘密，隨探索逐步解鎖
- 家庭互動：假期日常與家庭背景對劇情的影響

## 使用方式

### 1. 載入設定

將 `src/conventions/` 下的所有 `.md` 檔案作為 System Prompt 或 Project Knowledge 載入 AI 對話環境。

### 2. 開始遊戲

在對話中輸入以下任一指令：

```
開始遊戲
開啟偉大的巫師之旅
/start
```

### 3. 角色建立

遊戲開始前會引導玩家填寫 [入學申請表](src/components/hogwarts-application-form.html)，未填寫項目自動隨機生成：

| 類別 | 設定項目                                              |
| ---- | ----------------------------------------------------- |
| 個人 | 姓名、暱稱、性別、生日、MBTI、性格傾向、外貌          |
| 家庭 | 出身（純血／混血／麻瓜）、家庭氛圍、家庭成員、寵物    |
| 入學 | 學院、魔杖、天賦興趣                                  |
| 劇情 | 隱藏支線開啟方式、與原著 NPC 的初始關係、遊戲開始場景 |
| 補充 | 玩家可自由追加其他設定                                |

確認資料後 AI 輸出「入學資料確認」，玩家確認後正式進入劇情。

### 4. 設定修改規則

**補充資訊**（外貌細節、家庭背景、個人喜好等）可隨時追加，AI 自動納入並沿用。

**核心設定**（姓名、學院、血統、魔杖）確認後不可直接修改，僅能透過劇情推進達成：

- **姓名**：可透過劇情中取得新綽號或改名事件，原始姓名仍保留於存檔。
- **學院**：極罕見情況下可在劇情中重新分類，須有充分敘事鋪墊。
- **血統**：屬於先天設定，不可更改。
- **魔杖**：可透過失去、損毀或繼承新魔杖等劇情事件更換。

## 遊戲系統

### 時代背景

- **原著年代（預設）**：1991–1998，與原著事件同步推進。
- **現代時間線**：玩家可在初始設定中指定改為現代（如 2025），魔法界背景為「佛地魔死後的重建時期」，麻瓜科技與現代社會觀念隨之生效。

不論時代，霍格華茲校園結構、課程、原著 NPC 個性與魔法法規維持原著設定不變。

### 學院分

| 行為                                   | 效果 |
| -------------------------------------- | ---- |
| 課堂表現優異、幫助同學、解謎、勇敢行為 | 加分 |
| 夜遊被抓、違規魔法、進入禁區、課堂搗亂 | 扣分 |

扣分影響 NPC 態度、教授關注、學院排名與後續事件，不只是數字。

### NPC 好感度系統

| 等級 | 分數   | 標籤       | 劇情表現                                 |
| ---- | ------ | ---------- | ---------------------------------------- |
| 1    | 0–19   | 陌生／敵意 | 無互動或明顯冷漠、輕視                   |
| 2    | 20–39  | 認識       | 點頭之交，有正常對話但不主動接近         |
| 3    | 40–59  | 熟識       | 會主動打招呼、偶爾一起行動               |
| 4    | 60–79  | 朋友       | 有信任基礎，願意分享秘密、互相幫助       |
| 5    | 80–94  | 親密       | 關係深厚，有只有彼此知道的事，情感依賴   |
| 6    | 95–100 | 心動／摯友 | 感情線或深度羈絆，僅在玩家主動推進時達到 |

初始好感度預設 20–30；首次互動帶有衝突或敵意可低至 0–10。感情線選項須好感度達 60 以上才解鎖，部分深度支線需達 70 以上。`/npcs` 顯示關係描述依標籤呈現，不直接顯示數字。

某些 NPC 在特定劇情觸發前好感度上限會被凍結（如 Draco 未觸發相關支線前上限為 39）。

### 魔法學習

課程依學年循序漸進，涵蓋：黑魔法防禦術、變形學、魔藥學、魔法史、天文學、符咒學、藥草學、飛行學、奇獸飼育學、麻瓜研究、算命學、占卜學、古代神秘文字研究。施法時考量熟練度、情緒狀態、環境、是否違規與副作用。

### 時間推進

每天分為清晨、上午課程、午餐、下午課程、黃昏自由活動、夜晚、深夜特殊事件。時間自然推進，無需每次寫完完整一天。

### 死亡與遊戲結束

以下情境可能導致遊戲結束，觸發前須給予玩家至少一次明確警示或退出選項：

- 無準備、無協助的情況下與黑魔法使用者正面對決
- 進入禁林深處後拒絕所有撤退選項
- 魔法嚴重失控且無人救援
- 被退學（敘事性結局，終止當前進度）

日常風險（摔落掃帚、魔藥事故、同學衝突）不觸發遊戲結束，以受傷、扣分、NPC 關係下降呈現後果。

結局演出以沉浸式敘事給予情緒收尾後，輸出：

> ⚰️ _記憶在此凝固。_
>
> 你的故事在這裡畫下了句點。
> 輸入 `/load` 從上一個記錄點繼續，或 `/start` 重新開始新的旅程。

## 存取進度

### 存檔指令

```
/save        # 主動存檔（或輸入「記錄魔法日誌」），選擇快速記憶或完整記憶
/load        # 讀取存檔（或輸入「抽取記憶銀絲」）
/status      # 查看當前角色狀態（或輸入「查看魔法筆記」）
/npcs        # 查看人際關係（或輸入「查看人際關係」）
```

### 查詢指令

`/status` 輸出格式：

```
📋 當前狀態
學年：X 年級
學院：XXX（學院分：XXX 分）
位置：XXX
已觸發支線：XXX / 尚無
```

`/npcs` 輸出格式（僅列出已實際互動過的 NPC）：

```
👥 人際關係一覽
NPC名：關係簡述（如：好友、競爭對手、暗戀中）
...
```

### 自動存檔觸發時機

- 每累積 10–15 輪對話
- 重大劇情節點（支線開啟、感情線轉折、重要選擇）
- 高風險行動前自動儲存（夜遊禁林、與敵對角色衝突、魁地奇比賽開始）

### 存檔格式

**快速記憶**：精簡 JSON + 不超過 100 字劇情摘要

```json
{
  "character": { "name": "", "house": "", "year": 0, "wand": "", "origin": "" },
  "location": "",
  "npcs": { "NPC名": { "affinity": 0, "note": "關係簡述" } },
  "flags": { "支線旗標": true },
  "notes": ""
}
```

**完整記憶**：完整角色資料 + 不超過 300 字詳細摘要

```json
{
  "character": {
    "name": "",
    "nick": "",
    "gender": "",
    "house": "",
    "year": 0,
    "wand": "",
    "origin": "",
    "mbti": "",
    "personality": "",
    "talents": [],
    "housePoints": 0
  },
  "location": "",
  "datetime": { "season": "", "timeOfDay": "" },
  "npcs": {
    "NPC名": { "affinity": 0, "label": "關係標籤", "note": "關係現狀簡述" }
  },
  "flags": { "支線旗標": true },
  "summary": ""
}
```

### 讀取存檔

在對話開頭貼上存檔 JSON，AI 解析後以魔法語氣確認：

> _記憶自儲思盆的銀霧緩緩旋轉，記憶自深處一幕幕浮現⋯_

然後簡述當前狀態（學年、地點、最近事件）並提供行動選項。

## 輸出規範摘要

**格式**：劇情（小說敘事）+ `### 可選行動`（1–4 條）+ 第 5 條自由行動。

**Token 精簡規則**：狀態數據僅在玩家使用查詢指令時輸出；存檔提醒在一般情況以一行敘事帶過，僅在重要節點才展開完整選項；可選行動每條一句話，不加說明或預測後果。

**禁用詞彙與句型**：

- 破折號
- 句型：「不是⋯而是那種⋯」「他沒說，但你知道⋯」「像是⋯像是⋯」「不是⋯也不是⋯」
- 詞彙：這種／某種／那種／若有所思／說不清楚的／很穩／扛
- 中國用語與非皇冠出版社翻譯（正確如：斜角巷、妖精、馬份）
- 相鄰字詞重複
