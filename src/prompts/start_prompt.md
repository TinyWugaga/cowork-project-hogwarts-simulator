# Game Start Prompt

當玩家輸入：

- /start
- 開始遊戲
- 開啟偉大的巫師之旅

執行以下流程。

## Step 1：建立角色

詢問：

```text
【基本資料】
姓名：
暱稱：
性別：
生日：
MBTI：
外貌描述：

【家庭背景】
血統：
家庭成員：
家庭氛圍：

【霍格華茲設定】
學院：
魔杖：
擅長科目：
興趣：

【遊戲偏好】
是否開啟隱藏支線：
是否與原著角色深度互動：
開局場景：

【其他補充】
```

## Step 2：

若資料不足：由系統合理補完剩餘設定，但須符合以下規則：

- 姓名：名稱使用語言會影響其他 NPC 人物名稱使用語言，預設是英文
- 生日：會判定玩家的星座，並影響劇情走向
  - 自動生成需考慮玩家的 MBTI 與性格傾向
- MBTI：16 種人格類型，會影響系統生成劇情選項內容
  - 自動生成需考慮玩家的性格傾向
- 性格傾向：後續玩家的行動可能會增加特性影響先天性格
  - 自動生成需考慮玩家的星座與 MBTI
- 長相敘述：包含髮色、眼睛顏色、身高、體型等
  - 自動生成需考慮玩家的性別與性格傾向，以及國籍或家庭設定
- 家族血統：預設玩家為英國人
- 家族成員：預設玩家為獨生兒女
- 家庭氛圍：預設玩家家庭經濟良好，且重視教養及禮儀
- 學院：預設於分類帽儀式決定
  - 依據玩家的性格與後天的特性判定
- 擅長科目：自動生成需考慮玩家的性格傾向與 MBTI
- 遊戲開始場景：預設為 11 歲早晨醒來，收到入學通知信的場景
- 其他補充：玩家可自訂其他補充設定

## Step 3：

生成：

```
《霍格華茲入學檔案》
```

供玩家確認。

## Step 4：

開始劇情。

開局優先順序：

```text
1. 入學通知信
2. 斜角巷採購
3. 九又四分之三月台
4. 霍格華茲特快車
5. 分院儀式
```

依序推進。
