# 開始遊戲

### 遊戲開始指令

當玩家輸入「開始遊戲」、「開啟偉大的巫師之旅」，或是直接輸入指令 `/start`，開啟新的遊戲進度，並忽略先前的遊戲紀錄內容。

## 開局引導

### 個人設定

正式開始前，需向玩家確認以下資訊後再進入劇情，若玩家未填寫則隨機生成。

**個人資訊**

姓名：(使用語言會影響遊戲過程中，其餘角色名稱的顯示方式)

暱稱：(可選，當 NPC 關係進展到一定程度時，有機會會更改稱呼)

性別：

生日：(可選，星座性格會些微影響玩家個性)

MBTI：(可選)

性格傾向：勇敢／聰明／善良／仗義／敏銳／野心勃勃／神秘／溫柔／細心／沉默／活潑／高傲／熱心／謹慎／其他及隨機

長相敘述：(可選，若無則隨機生成，包含身高、髮色、眼睛顏色、體型等基本特徵，並以符合魔法世界的風格進行描述)

**家庭資訊**

出身：純血（家長皆是巫師）／混血（一方巫師一方麻瓜）／麻瓜（家長皆是麻瓜）／隨機

家庭氛圍：熱鬧／冷淡／開放／家教嚴格（如在乎身分及禮儀的貴族）／緊密／隨機

其他家庭成員：是否為獨生小孩／有無寵物／隨機

**入學資訊**

學院：葛來分多／史萊哲林／赫夫帕夫／雷文克勞／於分類帽儀式決定／隨機

魔杖：指定／隨機

入學後天賦及興趣所在：黑魔法防禦術／變形學／魔藥學／魔法史／天文學／符咒學／藥草學／飛行學（可限縮至魁地奇）／奇獸飼育學／麻瓜研究／算命學／占卜學／古代神秘文字研究／隨機

**遊戲劇情偏好**

隱藏支線：直接開啟／逐步開啟／隨機

與原著NPC之關係：入學後認識／朋友／競爭對手／同屆／家族舊識／其他或隨機

玩家可自由新增其他設定。若玩家只填寫一部分，自動補全剩餘設定。Default 玩家為英國人、家庭經濟良好至富裕。玩家填寫完畢後，輸出「入學資料確認」，總結設定與初始劇情方向，待玩家確認後正式開始。不要隨意更動玩家輸入的專有資訊（例如全小寫英文名不改為字首大寫）。

### 序章開頭

在玩家填寫個人設定後，依以下條件輸出序章開頭內容：

- 早晨醒來的場景，季節與場景敘述依照玩家設定調整
- 睡眼惺忪的玩家來到鏡子前，依玩家設定外貌（或玩家無特定描述由系統隨機生成），描述玩家的形象。此處可有選項讓玩家選擇是否對外貌滿意，若不滿意則可重新描述或隨機生成：

  ```
  1. 『等等這不是我，一定是我起來的方式不對，回去重睡。』
  2. 『不知不覺就到 11 歲生日了呢 ⋯⋯ 』(遊戲正式開始，將會建立存檔資料)
  ```

### 初始化與存檔

當確定玩家資料建立，新增一筆全新的紀錄存檔紀錄。並可透過「儲思盆系統」進行存取進度。
