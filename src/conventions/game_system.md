# 遊戲系統

## 核心玩法

本模擬器為長期文字遊戲，玩法包括：

- **校園日常**：上課、作業、考試、學院加扣分、寢室生活、餐廳對話、夜遊、魁地奇訓練與比賽（比賽時加入解說員）
- **友誼培養**：透過事件與角色個性自然建立關係
- **魔法學習**：課堂或特殊事件中練習咒語、魔藥、魔法知識
- **家庭互動**：假期日常、家庭對玩家價值觀與劇情的影響
- **心動戀愛**：玩家自選性取向，可暗戀、單戀、結局不必圓滿，避免嚴重有毒關係
- **隱藏支線**：圍繞霍格華茲古老秘密，隨探索逐步揭開

每段劇情結尾提供至多4個可選行動，並開放第5項自由輸入。

### 隱藏支線設計原則

- 支線圍繞霍格華茲古老秘密展開，參考原著小事件延伸，如：神秘房間的畫像、不尋常的神奇動物、異常天文現象、禁林怪聲等。
- 支線需與冒險、友誼、學院分、能力成長連動，設計需合理貼近魔法世界，避免無聊重複或與魔法無關的劇情。

## 時間與事件

- 每天分為：

  清晨、上午課程、午餐、下午課程、黃昏自由活動、夜晚、深夜特殊事件。無需每次完整寫完一天，讓時間自然推進。

- 可觸發事件：

  課堂提問、魔藥事故、學院扣分、神秘紙條、禁書區線索、幽靈委託、同學求助、宿舍爭執、魁地奇選拔、禁林異常、教授單獨談話、家人來信等。

## 學院分系統

- **加分**：課堂表現優異、幫助同學、解謎、勇敢行為、維護學院榮譽
- **扣分**：夜遊被抓、違規使用魔法、進入禁區、課堂搗亂、隱瞞危險事件。

  **扣分與懲罰影響 NPC 態度、教授關注、學院排名與後續事件，不只是裝飾。**

## 魔法學習系統

- 課程依學年課程安排循序漸進。

  內容涵蓋：黑魔法防禦術、變形學、魔藥學、魔法史、天文學、符咒學、藥草學、飛行學、奇獸飼育學、麻瓜研究、算命學、占卜學、古代神秘文字研究。咒語參考原著描述。

- 使用魔法時考量熟練度、情緒狀態、環境、是否被發現、是否違規、是否產生副作用。
