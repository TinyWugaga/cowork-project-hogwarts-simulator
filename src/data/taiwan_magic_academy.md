# 台灣魔法學校介紹

## 🇹🇼 雲天宮（非官方創作設定）

> 參考《哈利波特》世界觀，結合台灣歷史、民間信仰、道教文化與原住民族傳說所設計。

### 學校簡介

雲天宮是一所隱藏於台灣中央山脈深處的古老魔法學院。

其名稱取自：

- 雲海
- 北港朝天宮

其教育理念融合：

- 自然共生
- 靈性修行
- 符術傳承
- 守護結界

與西方巫師強調個人魔力不同，雲天宮更重視「借天地之力」。

### 地理位置

- 位於玉山或中央山脈深處。
- 被大型結界與迷霧法陣保護。
- 麻瓜僅能看見高山茶園與雲霧。

### 校園建築

主要建材：

- 台灣紅檜
- 大理石
- 山林靈木

特色：

- 宮殿式建築群
- 茶園梯田
- 雲海觀星台
- 巨型護山法陣

### 通勤文化

#### 新生交通工具

學生可透過：

- 巨型帝雉
- 幽靈區間車
- 傳送結界站

往返學校。

#### 校園作息

- 早自習
- 晚自習
- 符術演練
- 陣法修習

### 魔法夜市

校園周邊設有著名的魔法夜市。

#### 魔法臭豆腐

效果：

- 產生驅邪煙霧

#### 魔藥珍珠奶茶

依配方不同可提供：

## 桃木劍施法體系

### 核心法器

雲天宮並不使用西方魔杖。

每位學生於入學後都會獲得一柄專屬的：

- 桃木劍
- 法劍
- 鎮靈劍

其地位相當於霍格華茲巫師的魔杖。

### 為何選擇桃木劍

在台灣魔法界認為：

> 桃木天生具有驅邪、鎮煞、破幻與淨化的力量。

因此特別適合作為施法媒介。

### 桃木劍結構

#### 劍柄

刻有個人符印：

- 北斗七星
- 八卦
- 雷紋
- 家族徽記
- 原住民族圖騰

#### 劍身

刻滿微型符文。

作用類似西方魔杖內部的魔法迴路。

#### 劍穗

代表：

- 年級
- 專精方向
- 特殊資格

### 施法方式

西方巫師：

- 揮動魔杖
- 念誦咒語

雲天宮學生：

- 揮動桃木劍
- 描繪符印
- 念誦咒文

施法三要素：

1. 咒語
2. 劍勢
3. 符印

缺一不可。

### 法劍評鑑制度

桃木劍會隨著修行成果產生變化。

| 狀態     | 象徵       |
| -------- | ---------- |
| 淺木色   | 初學者     |
| 金色紋路 | 優秀學生   |
| 玉石光澤 | 頂尖畢業生 |
| 黑色裂紋 | 使用禁術者 |

#### 玉桃劍

當法劍出現玉石般光澤時，代表獲得最高榮譽。

被稱為：

> 玉桃劍

象徵已達到符劍合一的境界。

### 禁術標記

若學生：

- 使用禁咒
- 修習邪術
- 濫用靈體力量

法劍將出現黑色裂痕。

此狀態會被學校與魔法部列入監管。

## 課程特色

### 劍訣學

相當於霍格華茲的魔咒學。

內容包括：

- 劍勢運用
- 咒語控制
- 靈力導引

### 符籙學

必修課程。

教授：

- 驅邪符
- 隱身符
- 定身符
- 治療符
- 結界符

### 靈媒學

結合台灣民間的起乩文化。

可召喚靈體，以自身為媒介，與高階靈體、守護神、祖靈或山林精怪進行溝通，獲得指引。

#### 主要課程

- 問卦解籤
- 分辨靈體種類
- 起乩降駕

### 陣法學

台灣魔法界最具代表性的科目。

學生必須合作完成：

- 守護法陣
- 驅魔法陣
- 傳送法陣
- 山林結界

### 靈獸學

研究本土奇獸與守護靈。

## 奇獸與魔藥學

### 石虎

能力：

- 看穿幻術
- 偵測隱形生物

被視為優秀的魔法夥伴。

### 台灣黑熊

能力：

- 破除迷陣
- 山林穿行
- 驅散邪靈

稀有奇獸，也被稱為「山林守護者」。

### 魔神仔

危險等級較高。

能力：

- 迷惑心智
- 引導迷路
- 製造幻覺

為防禦術的重要教材。

### 魔法涼茶學

結合青草藥文化。

常見材料：

- 咸豐草
- 艾草
- 山苦瓜
- 萬年青

用途：

- 解毒
- 解咒
- 驅邪
- 抵禦黑魔法

## 核心價值

雲天宮培養的並非單純的戰鬥巫師，而是守護者。

其教育精神為：

- 天人合一
- 敬畏自然
- 守護眾生
- 傳承文化
- 群體合作

因此在國際魔法界中，雲天宮最著名的領域並非決鬥，而是：

- 結界術
- 驅邪術
- 群體防護法陣
- 靈體溝通
- 山林守護魔法
